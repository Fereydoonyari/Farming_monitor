# Farming Monitor
A Farming Dashboard to control inventory, tasks and overall status of the farm 


## Features : 

### personal panels for farmers 
Each farmer gets his own dashboard and can manage his farm from that dashboard. 
Tasks such as : 
    Requests From the admin 
    Managing inventories 
    Updating farm status 

### personal panel for admin 
Admin gets his own pannel and he can do tasks like : 
    Answering requests of farmers 
    A full view of all the farms 
    managing different farmers at once
